# Analysis Data Dictionary

The clean analysis files are generated by running:

```powershell
python ingest.py
```

## `fars_crashes_2019_2023.csv`

One row per fatal crash from NHTSA FARS. Important fields:

- `case_id`: FARS crash identifier.
- `fatalities`: Number of fatalities in the crash.
- `latitude`, `longitude`: Reported crash coordinates.
- `commercial_vehicle_involved`: `1` when FARS records a motor-carrier identifier.
- `large_truck_bus_involved`: `1` when a large-truck or bus body/configuration is present.

Neither commercial field proves a person was working when the crash occurred.

## `county_target_industries_2023.csv`

One row per county and target NAICS industry from Census County Business Patterns.
Target industries are specialty trade contractors, truck transportation, couriers and
messengers, and landscaping services.

## `county_population_2020_2024.csv`

One row per US county containing annual Census population estimates.

## `county_marketing_analytics.csv`

One row per county and FARS year. This joins fatal-crash outcomes, Census population,
and target-industry market size.

- `fatal_crashes_per_100k`: Fatal crashes divided by population, multiplied by 100,000.
- `fatalities_per_100k`: Fatalities divided by population, multiplied by 100,000.
- `commercial_share`: Share of fatal crashes with a recorded motor-carrier identifier.
- `target_establishments`: Establishments in the four selected target industries.
- `target_industry_employees`: Employees in those industries; suppressed Census values
  may be missing.

Population estimates begin in 2020, so 2019 per-capita rate fields are intentionally
missing. Small-county rates can be volatile; apply a reasonable population threshold
before using them for marketing decisions.

Treat this table as an EDA foundation, not a final marketing recommendation model.

## `utah_crashes_2019_2024.csv.gz`

One row per anonymized UDOT-reported Utah crash from completed calendar years
2019-2024. It includes street and route information, crash coordinates, severity,
injury counts, motor-carrier and commercial-vehicle indicators, weather, lighting,
collision type, DUI, distraction, speed, pedestrian, bicyclist, and motorcycle fields.

The file is gzip-compressed because it contains more than 350,000 records. R's
`readr::read_csv()` reads it directly without manual extraction.

## `utah_county_crash_analytics_2019_2024.csv`

The recommended starting table for Utah EDA. It contains exactly one row for each
of Utah's 29 counties in each year from 2019-2024.

Important fields include:

- `all_crashes` and `all_crashes_per_100k`
- `injury_crashes` and `injury_crashes_per_100k`
- `suspected_serious_injury_crashes`
- `fatal_crashes`
- `motor_carrier_crashes` and `commercial_vehicle_crashes`
- `dui_crashes`, `distracted_driving_crashes`, and `speed_related_crashes`
- `target_establishments` and `target_industry_employees`

Population-adjusted fields are unavailable for 2019 because the loaded Census
population-estimate series begins in 2020.
